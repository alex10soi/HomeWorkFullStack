Перед началом работы необходимо настроить работу сервера и доступ к нему, поэтому следует выполнить следующие действия:
1) Найдите файл 'configDataServer.php' в папке Chat-MySQL\resources\php ;
2) Откройте файл 'configDataServer.php' в редакторе и впишите свои данные в поля переменных констант, таких как:  
	 'SERVER_NAME' (серверное имя),
	 'USER_NAME' (имя пользователя),  
	 'MYSQL_PASSWORD' (пароль пользователя).;
3) Сохраните изменения (Ctrl + S) и закройте файл;
4) Есть два способа стартовать программу. Используйте первый способ, а в случае неудачи переходите ко второму способу.
	Первый способ :
	- запустите файл 'connectMySQL.php', после запуска этого файла база данных и содержимое в ней будет создано автоматически 
	  о чем будет выведено сообщение на экран об успешном создании всего или выведет ошибки;
	- если после всего этого не была создана база данных 'easychat' с таблицой 'registrinfo' и 'chatinfo', тогда переходите
	  ко второму способу (см. ниже);
	- запустите файл 'index.php', который вы найдете в папке Chat-MySQL. Программа запущена);

	Второй способ :
  - Зайдите в PhpMyAdmin, авторизуйтесь, после чего вам следует импортировать файл 'easychat.sql' тут же в phpMyAdmin
	  c помощью опции 'импорт'(import), вышеуказанный файл вы можете найти в папке 'sql';
	  В следствии чего будет создана база данных 'easychat' включающая в себя две таблицы 'registrinfo' (для хранения данных
	  пользователя взятых с регистрационной формы чата) и 'chatinfo' (для данных пользовательской переписки);
